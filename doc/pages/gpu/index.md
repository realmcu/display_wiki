# GPU 图形处理器

本章节深入介绍图形处理器（GPU）技术，涵盖 2D、2.5D 和 3D GPU 的架构对比、渲染管线、内存开销分析等内容。

图形处理器是现代显示系统的核心组件，负责高效处理图形渲染任务。根据功能复杂度，GPU 可分为三类：

- **2D GPU**: 平面图形加速，主要用于 UI 渲染和图层合成
- **2.5D GPU**: 支持矢量图形和变形效果，提供"伪 3D"能力
- **3D GPU**: 完整的三维渲染管线，支持光照、纹理映射和可编程着色器

**学习路径建议**: 初学者建议从 GPU 技术对比开始，然后深入学习 3D GPU 渲染管线，最后参考内存开销分析进行系统设计。
